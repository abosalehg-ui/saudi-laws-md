# laws/ — نصوص الأنظمة

هذا المجلد يضم نصوص الأنظمة واللوائح السعودية بصيغة Markdown، مصنّفة حسب المجال/النوع.

## الترخيص والنطاق

- **النصوص هنا (`laws/`)** تُعد ملكًا عامًا في المملكة العربية السعودية بموجب أحكام نظام حماية حقوق المؤلف السعودي، وليست مشمولة برخصة كود الأداة.
- **كود الأداة (`scripts/`)** مرخّص بموجب [MIT](../LICENSE).

## تنويه

نسخ **غير رسمية** مستخرجة آليًا من [qanoonsa.com](https://qanoonsa.com) و[nezams.com](https://nezams.com). لا يُعتد بها كمصدر رسمي؛ للتحقق يُرجى الرجوع إلى [جريدة أم القرى](https://uqn.gov.sa) و[بوابة الأنظمة السعودية لدى هيئة الخبراء](https://laws.boe.gov.sa).

## الفهرس الآلي

[`index.json`](../index.json) (في جذر المستودع) يوفّر فهرسًا آليًا بالبيانات الوصفية لكل وثيقة (العنوان، النوع، التصنيف، الحالة، المصدر، الرابط، المسار).
