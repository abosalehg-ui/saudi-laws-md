# saudi-laws-md

نصوص الأنظمة واللوائح السعودية بصيغة Markdown نظيفة، منظمة حسب المجال، للبحث والاستخدام في أدوات المطورين والباحثين.

المستودع: https://github.com/abosalehg-ui/saudi-laws-md

## ⚠️ تنويه مهم

هذا المستودع **غير رسمي** وغير تابع لأي جهة حكومية، وهو تجميع شخصي لنصوص الأنظمة السعودية.

**لا يُعتد بهذا المستودع كمصدر رسمي.** للتحقق من دقة أي نص أو معرفة آخر التعديلات، يُرجى الرجوع إلى:
- [جريدة أم القرى الرسمية](https://uqn.gov.sa)
- [بوابة الأنظمة السعودية (هيئة الخبراء بمجلس الوزراء)](https://laws.boe.gov.sa)

## المحتوى

- تصنيف الأنظمة حسب المجال (جرائم، تجاري، عمل، إداري... إلخ) داخل `laws/`
- كل نظام بملف Markdown مستقل يحتوي:
  - بيانات الإصدار (المرسوم/القرار، التاريخ الهجري والميلادي، رقم عدد أم القرى)
  - النص الكامل مقسّمًا حسب الأبواب والمواد
  - سجل التعديلات التاريخية لكل مادة (عند توفره)

## البنية

```
saudi-laws-md/
├── laws/
│   ├── جرائم/
│   ├── تجاري/
│   ├── عمل/
│   └── ...

```

## صيغة الملف

كل نظام يتبع القالب التالي:

```markdown
---
title: اسم النظام
issued_by: ...
approval_date_hijri: ...
publish_date: ...
gazette_ref: ...
status: نافذ / معدّل
category: ...
attachments: [...]
---

# اسم النظام

## المادة الأولى
...

> **تعديلات المادة:**
> - ...
```

قسم "تعديلات المادة" يظهر فقط عند توفر سجل تعديلات تاريخي لتلك المادة.

## التعديلات على الأنظمة

بعض الأنظمة تصدر بنسخ معدّلة لاحقًا. يحتفظ المستودع بالنسخة الحالية في `laws/`، وبالنسخ السابقة (إن وُجدت) في `history/` مع الإشارة إلى تاريخ التعديل ورقم القرار المعدِّل.

## المساهمة

المساهمات مرحّب بها، خصوصًا في:
- تصحيح أخطاء (ترقيم، تنسيق، مواد ناقصة)
- إضافة أنظمة جديدة
- تحسين التصنيف

يُرجى فتح Issue قبل أي تعديل جوهري على البنية.

## الترخيص

نصوص الأنظمة تُعد ملكًا عامًا في المملكة العربية السعودية بموجب أحكام نظام حماية حقوق المؤلف السعودي.
